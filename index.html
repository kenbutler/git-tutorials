Let's Learn Git!

Git is a distributed version control system that allows software developers to track their development history and supports distributed team development. While other resources for learning Git lean towards either textual or command line format, this site aims at teaching Git through tutorials, scenarios, and parallel tracks for both the command line interface and GUI's.

Visit the links in the top menu to get started. You may also select the down arrow to view a content breakdown, and the right arrow to view additional resources for learning Git.
