What is Git?

Git is an...
  open-source,
  distributed,
  version control system
  designed for speed and efficiency
  that supports multiple workflow types

Version Control

Version control is a system that records changes to a file or set of files over time, so that you can recall specific versions later
- Git-SCM

Think of version control as a time machine that let's you look at history and go forwards and backwards though it!

Open-Source

“Git is released under the GNU General Public License version 2.0, which is an open source license. The Git project chose to use GPLv2 to guarantee your freedom to share and change free software---to make sure the software is free for all its users.” Git SCM - About
